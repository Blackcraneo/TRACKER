INSTRUCCIONES PARA GENERAR TOKEN OAUTH DE TWITCH
==================================================

1. PRIMERO: Actualizar URL de redirección en Twitch Dev Console
   - Ve a: https://dev.twitch.tv/console
   - Selecciona tu aplicación "BlackcraneoTracker"
   - En "URL de redireccionamiento de OAuth", agrega:
     http://localhost:3000
   - Haz clic en "Añadir"
   - Haz clic en "Guardar"

2. SEGUNDO: Generar el token OAuth
   - Copia esta URL y pégala en tu navegador:
   
https://id.twitch.tv/oauth2/authorize?client_id=mo983ad8zpisqtkezy4q4ky7qvcoc4&redirect_uri=http%3A%2F%2Flocalhost%3A3000&response_type=token&scope=user%3Aread%3Aemail+moderator%3Aread%3Achatters+channel%3Aread%3Asubscriptions+channel%3Aread%3Astream_key+user%3Aread%3Abroadcast&state=tracker_auth

3. TERCERO: Autorizar la aplicación
   - Twitch te pedirá autorización
   - Haz clic en "Autorizar"
   - Serás redirigido a una página de error (normal)
   - Mira la barra de direcciones del navegador

4. CUARTO: Extraer el token
   - En la URL del navegador, busca: access_token=
   - Copia todo el texto después de "access_token=" hasta el primer "&"
   - Ejemplo: si la URL es:
     http://localhost:3000/#access_token=abc123&token_type=bearer&scope=...
     El token es: abc123

5. QUINTO: Actualizar el archivo .env
   - Crea o edita el archivo .env en la carpeta TRACKER
   - Agrega esta línea:
     TWITCH_OAUTH=tu_token_aqui
   - Reemplaza "tu_token_aqui" con el token que copiaste

6. SEXTO: Reiniciar la aplicación
   - Detén la aplicación (Ctrl+C)
   - Ejecuta nuevamente: python app.py
   - Ve a http://localhost:3000
   - Los logs deberían mostrar "Token valido" y "Canal encontrado"

SCOPES INCLUIDOS:
- user:read:email
- moderator:read:chatters
- channel:read:subscriptions
- channel:read:stream_key
- user:read:broadcast

NOTAS:
- Como owner del canal, tienes todos los permisos necesarios
- El token expira, necesitarás renovarlo periódicamente
- Para producción, usa la URL de Railway en lugar de localhost
